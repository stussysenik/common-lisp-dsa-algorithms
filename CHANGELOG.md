# Changelog

## 0.1.0 - Initial Release
- Dynamic array, linked list, stack, queue, deque
- Binary heap, priority queue
- Hash table (open addressing), LRU cache
- Binary tree, BST, AVL tree, Red-Black tree
- Trie, segment tree
- Graph (adjacency list), DAG, union-find
- Bloom filter, skip list
- Sorting: bubble, selection, insertion, merge, quick, heap, counting, radix
- Searching: linear, binary, jump, interpolation
- Graph algos: BFS, DFS, Dijkstra, A*, Bellman-Ford, Floyd-Warshall, Prim, Kruskal
- DP: fibonacci, knapsack, LCS, LIS, edit distance, coin change
- String: KMP, Rabin-Karp, Z-algorithm, Manacher, LCS substring
